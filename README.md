## Note

### For Mac Users:
 You might encounter the error:


```py
ImportError: No module named cv2
```

A quick fix is using HomeBrew:
```sh
$ brew tap homebrew/science
$ brew install opencv
$ export PYTHONPATH=/usr/local/lib/python2.7/site-packages:$PYTHONPATH
```
